dados-tse-graphql
=======================

Backend em [GraphQL](https://graphql.org/) para facilitar obter os dados divulgados pelo [TSE](http://www.tse.jus.br/) dos candidados.

### Como iniciar o app

```bash
npm i
npm start
```

Em `http://localhost:4000` estara um webapp para fazer queries e o endpoint da api.
